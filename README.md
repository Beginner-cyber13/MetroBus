🚍 Metro Bus Routing & Ticketing System (Twin Cities)

A web-based Metro Bus management system developed as a semester project to simplify public transport navigation and ticketing in Islamabad & Rawalpindi.

This system helps commuters explore routes, understand available bus types, and simulate a ticketing workflow in one centralized platform.

📌 Project Overview

Public transport users often struggle to find clear routing information and ticketing details for different metro bus services.
This project provides a centralized routing and ticketing system that allows users to:

View available bus routes

Explore different bus types

Simulate ticket booking

Access organized transport information in one place

🧠 Key Features

Bus route discovery and navigation

Support for 4 different bus types

Ticket booking workflow simulation

Database-driven system for route and ticket data

User-friendly web interface

🛠️ Tech Stack
Layer	Technology
Backend	ASP.NET MVC, C#
Frontend	HTML, CSS
Database	SQL Server
Architecture	MVC (Model–View–Controller)
💻 System Modules
1️⃣ Routing Module

Displays available bus routes

Provides route details and bus categories

Helps users choose suitable travel options

2️⃣ Ticketing Module

Simulates ticket booking process

Stores and retrieves ticket information

Connects frontend with database using MVC architecture

3️⃣ Database Module

Stores routes, buses, and ticket data

Handles CRUD operations

Ensures efficient data retrieval and storage

⚙️ Challenges & Learning Outcomes

One of the biggest challenges in this project was:

This project significantly improved my skills in:
